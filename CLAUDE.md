# Project Instructions

## On Session Start
Run the health check script at the beginning of each session:
```
python health_check.py
```

## Project Overview
Tradovate futures trading bot using ICT (Inner Circle Trader) strategy.

## Key Commands
- `python health_check.py` - Run project health check
- `python -m pytest tests/` - Run all tests
- `python -m runners.run_replay` - Run backtest replay
